== Description ==

If you want to put your twitter account statistics in our wordpress site, this wp-vm-show-tweets plug-in does for you
the smae with multipl options and also you can have your own color combinatios for the widget, on the fly widget preview.
cache management to handle multiple requests and this also under your control. 


== Installation ==

There are two ways of installing the plugin:

**From the [WordPress plugins page](http://wordpress.org/extend/plugins/)**

1. Download the plugin
2. Upload the `wp-vm-show-tweets` folder to your `/wp-content/plugins/` directory.
3. Active the plugin in the plugin menu panel in your administration area.

**From inside your WordPress installation, in the plugin section.**

1. Search for wp admin custom fileds plugin
2. Download it and then active it.

Once, you have the plugin activated, you will find a new option called 'VM Tweet Plus Settings' under your wordpress Settings menu. 
There you have an dynamic interface to manage your twitter widget as you wish with lot of options.

Also You can find simple tutorials with video on the [WP VM Tweet Plus Settings plugin web page](http://blog.vinmatrix.com/products/wp-plugins/wp-vm-show-tweets/)
